# Kracken Messenger

## Исправления, которые были внесены

### 1. Улучшенное меню
- Заменено скриптовое меню на CSS-стилизованное
- Добавлены навигационные вкладки: "Chats" и "Friends"
- Современный дизайн с градиентами и backdrop-filter эффектами
- Адаптивный дизайн для мобильных устройств

### 2. Система друзей
- Добавлена полная система друзей с запросами
- Возможность отправлять и принимать запросы в друзья
- Отображение списка друзей и статуса отношений
- Уведомления о новых запросах

## Запуск проекта

### 1. Запуск сервера
```bash
cd apps/server
npm install
npm start
```
Сервер запустится на порту 3000

### 2. Запуск веб-версии
```bash
cd apps/web
npm install
npm start
```
Веб-версия запустится на порту 3001

### 3. Открыть в браузере
- Сервер: http://localhost:3000
- Веб-версия: http://localhost:3001

## Тестирование системы друзей

1. Откройте веб-версию в двух разных вкладках браузера
2. Войдите под разными аккаунтами (например, "nihma" и "Dev")
3. Переключитесь на вкладку "Friends"
4. Нажмите "+" для добавления друга
5. Введите имя пользователя и отправьте запрос
6. Во второй вкладке примите или отклоните запрос

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js, Express, Socket.IO
- **Database**: PostgreSQL/SQLite
- **Styling**: CSS Variables, Flexbox, CSS Grid
- **Real-time**: WebSocket через Socket.IO
