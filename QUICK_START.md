# 🚀 Быстрый запуск админ панели

## 🚨 Решение проблемы CORS

**Проблема:** При открытии HTML файла напрямую возникает ошибка CORS.

**Решение:** Используйте HTTP сервер или локальную версию.

## 🔧 Вариант 1: HTTP сервер (рекомендуется)

### 1. Запуск сервера
```bash
cd apps/server
npm start
```

### 2. Открытие админ панели
Откройте в браузере:
```
http://localhost:3000/admin
```

## 🔧 Вариант 2: Локальная версия

### 1. Откройте файл
Откройте `admin-panel-local.html` в браузере

### 2. Настройте сервер
Введите URL вашего сервера:
- Локальный: `http://localhost:3000`
- Продакшн: `https://krackenx.onrender.com`

## 🧪 Тестирование API
```bash
node test-admin-api.js
```

## 🎯 Готово!
Теперь у вас есть админ панель с функциями:
- ✅ Отправка сообщений от поддержки
- ✅ Удаление всех сообщений
- ✅ Удаление всех пользователей

## ⚠️ Внимание
Панель не защищена паролем! Используйте только в доверенной среде.

## 🔍 Файлы
- `admin-panel.html` - для сервера
- `admin-panel-local.html` - для локального использования
- `test-admin-api.js` - для тестирования
