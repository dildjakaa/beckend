# 🔄 Инструкция по перезапуску сервера

## ⚠️ Важно!
После внесения изменений в код **ОБЯЗАТЕЛЬНО** перезапустите сервер!

## 🚀 Шаги для перезапуска:

### 1. Остановите текущий сервер
```bash
# Если сервер запущен в терминале, нажмите Ctrl+C
# Или найдите процесс и остановите его
ps aux | grep node
kill -9 <PID>
```

### 2. Перейдите в папку сервера
```bash
cd apps/server
```

### 3. Установите зависимости (если нужно)
```bash
npm install
```

### 4. Запустите сервер заново
```bash
npm start
# или
node server.js
```

### 5. Проверьте логи
```bash
# В логах должно появиться:
✅ Friends system tables and indexes created/verified
```

## 🔍 Проверка работоспособности:

### Тест API endpoints:
```bash
node test-friends-api.js
```

### Проверка в браузере:
1. Откройте `http://localhost:3000`
2. Войдите в систему
3. Проверьте консоль браузера на ошибки
4. Попробуйте добавить друга

## 🐛 Если что-то не работает:

### Проверьте:
- [ ] Сервер запущен и работает
- [ ] В консоли нет ошибок
- [ ] База данных доступна
- [ ] Все файлы сохранены

### Логи ошибок:
```bash
# В консоли сервера должны быть:
✅ Friends system tables and indexes created/verified
✅ Server running on port 3000
```

## 📱 Тестирование мобильного интерфейса:

1. Откройте браузер
2. Нажмите F12 (DevTools)
3. Нажмите на иконку мобильного устройства
4. Выберите размер экрана ≤ 768px
5. Проверьте мобильную навигацию

## 🔧 Полезные команды:

```bash
# Проверка статуса сервера
curl http://localhost:3000/api/health

# Проверка логов
tail -f server.log

# Перезапуск с nodemon (если установлен)
nodemon server.js
```

---

**Помните**: После любых изменений в коде сервер нужно перезапускать!
